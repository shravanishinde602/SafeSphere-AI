# SafeSphere AI

SafeSphere AI is a smart web application that protects users from phishing websites and suspicious emails. It analyzes URLs and email content to provide instant verdicts with confidence scores, helping users stay safe online.

## Features
- Detect phishing and suspicious URLs  
- Identify typosquatting and brand impersonation  
- Scan emails for phishing patterns and urgency keywords  
- User-friendly, responsive interface  
- Instant results with confidence scores and explanations  


## Technologies
- Frontend:HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB, Mongoose  
- Deployment: Netlify (frontend), Render (backend)  
- Libraries & Tools: Fetch API, CORS, dotenv  


## Live Demo
- Frontend: [SafeSphere AI on Netlify](https://harmonious-bunny-53bb4c.netlify.app)  
- Backend:[SafeSphere AI on Render](https://safe-sphere-ai.onrender.com)  


## How to Use
1. Open the frontend URL in a browser.  
2. Enter a URL or email content.  
3. Click Scan.  
4. View results: Safe, Suspicious, or Phishing with confidence scores.
